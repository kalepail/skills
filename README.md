# Kalepail Skills

Reusable AI-agent skills maintained by Kalepail.

## Skills

- `skills/agent-browser-webauthn` - drive passkey and Stellar smart account browser tests with `agent-browser` and Chrome DevTools virtual WebAuthn authenticators.

## Install

Clone this repository into your agent's skills directory, or install it with a compatible skills manager.

```bash
git clone https://github.com/kalepail/skills ~/.codex/skills/kalepail-skills
```

Then ask your agent to use the skill at:

```text
~/.codex/skills/kalepail-skills/skills/agent-browser-webauthn
```
